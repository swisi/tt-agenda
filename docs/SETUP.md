# Setup (v2)

## Voraussetzungen

- Python 3.12+
- Paketmanager: `pip`
- Optional: Docker

## Geplanter Standard-Setup (wird mit Codeaufbau finalisiert)

1. Virtuelle Umgebung erstellen
2. Dependencies installieren
3. `.env` konfigurieren
4. Datenbank initialisieren
5. App starten

## Konfiguration

Geplante Kernvariablen:

- `APP_ENV`
- `SECRET_KEY`
- `DATABASE_URL`
- `LOG_LEVEL`
